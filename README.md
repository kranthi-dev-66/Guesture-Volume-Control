# 🎛️ Gesture Volume Controller

Control your system volume using hand gestures in real-time.

---

## 🚀 Features

* ✋ Hand tracking with MediaPipe
* 🎥 Real-time webcam feed (OpenCV)
* 🔊 Volume control using finger distance
* 📊 Live UI with volume & distance
* 🌙 Clean dark interface

---

## 🛠️ Tech Stack

Python • Flask • OpenCV • MediaPipe • NumPy • Pycaw

---

## ⚙️ Setup

```bash
git clone https://github.com/kranthi-dev-66/Gesture-Volume-Control.git
cd GVC
pip install -r requirements.txt
python app.py
```

---

## 🎮 Usage

1. Click **Start Camera**
2. Show your hand
3. Adjust thumb & index finger distance
4. Volume changes automatically
